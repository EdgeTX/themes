# Themes directory for third party themes for EdgeTX

For older, EdgeTX v2.5 compatible themes, please [go to the 2.5 branch](../../tree/2.5).

Browse and preview all available themes in the [EdgeTX Theme Gallery](https://edgetx.github.io/themes/).

[Click here](https://github.com/EdgeTX/themes/releases/download/latest/edgetx-themes.zip) to download the latest themes package which you can then copy onto your radio's SD card. Alternatively, check the [latest release](https://github.com/EdgeTX/themes/releases/tag/latest) page for recent changes.

## Online theme creators and templates

You have an option to create themes with easy-to-use online theme file creators or templates:
- [EdgeTX Theme yml thingie](https://qwad.se/edgetx-theme-yml-thingie/) by Mattias Kallio
- [EdgeTX Theme Photoshop Templates](https://github.com/GhostBranchFPV/themes/tree/main/Photoshop%20Files) by GhostBranchFPV
- [EdgeTX Theme Designer](https://edge-tx-theme-creator.vercel.app/) by [Giancarlo Monastero](https://github.com/giancarlomonastero-ui/EdgeTX-Theme-Creator)

These tools let you play with theme colours and generate the required files.

The EdgeTX Theme Creator by edriskus is unfortunately no longer available.
